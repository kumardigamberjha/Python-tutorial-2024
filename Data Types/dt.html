Lets Understand Each Data Type with Example and Syntax

Data Type Description Example Explanation
Integer Whole numbers (positive, negative, or zero) 10, -23, 0 Integers can be positive, negative, or zero. They are used to represent whole numbers and cannot have decimal places.
Float Decimal numbers 3.14159, -1.23, 0.0 Floats are used to represent decimal numbers. They can be positive, negative, or zero.
Boolean True or False values True, False Booleans represent True or False values. They are used in conditional statements and logical operations.
String Sequence of characters enclosed in quotes "Hello, World!", 'Python is fun!', "" Strings are sequences of characters enclosed in quotes. They can be used to represent text, names, and other types of data.
List Ordered collection of different data types enclosed in square brackets [1, 2, 3, "apple", "banana"], [True, False, None], [] Lists are ordered collections of different data types enclosed in square brackets. They are used to store and access data in a specific order.
Tuple Immutable ordered collection of different data types enclosed in parentheses (1, 2, 3, "apple", "banana"), (True, False, None), () Tuples are immutable ordered collections of different data types enclosed in parentheses. They are similar to lists, but they cannot be modified after they are created.
Dictionary Key-value pairs enclosed in curly braces {"name": "John", "age": 30, "city": "New York"}, {"key1": True, "key2": 10, "key3": None}, {} Dictionaries are unordered collections of key-value pairs enclosed in curly braces. They are used to store and access data using keys.
Set Unordered collection of unique values enclosed in curly braces {1, 2, 3, "apple", "banana"}, {True, False, None}, set() Sets are unordered collections of unique values enclosed in curly braces. They are used to store and access unique data values.
